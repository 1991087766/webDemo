Git 提交代码方法

1. 查看需要更新的文件或文件夹 <br/>
    git status<br/>
2. 添加要上传的文件或文件夹<br/>
    git add 文件夹或文件的名称<br/>
3. 添加注释（必填）<br/>
    git commit -m "添加首页 或 修改了某某方法，必须要用双引号括起来"<br/>
4. 上传代码<br/>
    git push<br/>
5. 下载代码 （多人项目每次开始下载一下更新）<br/>
    git pull<br/>
6. 撤销本地对某个文件的修改（慎重使用！）<br/>
    git checkout 文件或文件夹<br/>
